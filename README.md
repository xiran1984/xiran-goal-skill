# xiran-goal

> 把任何目标拆成条件树，找到瓶颈，走最短可验证路径。

## 安装

```
https://github.com/xiran1984/xiran-goal-skill，安装这个skill
```

## 怎么用

```
/xiran-goal 我想在3个月内从零开始做一个能赚钱的AI工具产品
```

## 例子

问：`/xiran-goal 我如何在社会上最大程度地生存和繁殖？`

答：

**Final Goal**
在现代社会中，合法、可持续地最大化长期生存概率和后代繁衍成功率。

**Condition Tree**

```text
在社会中最大程度生存和繁殖
├─ 别被淘汰 ⚠️ 归零等于游戏结束
│  ├─ 不违法、不暴力、不高杠杆赌博
│  ├─ 不成瘾
│  │  ├─ 酒精、赌博、毒品
│  │  ├─ 色情、游戏、短视频
│  │  └─ 成瘾定义：无法自主停止即算
│  ├─ 守住身体底线
│  │  └─ 睡眠、体重、每年体检
│  └─ 6个月生活费现金缓冲
│
├─ 让社会持续需要你
│  ├─ 找到一个可变现技能
│  │  ├─ 有人已经在为它付钱
│  │  ├─ 你能做出可展示样品
│  │  └─ 6个月内可量化进步
│  ├─ 进入真实交易市场
│  │  └─ 不只在学，要卖
│  └─ 收入→抗风险资产
│     └─ 先存钱后投资，不靠消费证明自己
│
├─ 社会位置：让别人愿意把机会交给你
│  ├─ 说到做到
│  ├─ 做不到提前说
│  └─ 切断消耗型关系
│
├─ 择偶竞争力
│  ├─ 最低门槛：干净、稳定、不失控
│  ├─ 长期价值 ⚠️ 决定性瓶颈
│  │  ├─ 有责任感（不是嘴说，是持续做）
│  │  ├─ 能处理冲突（不逃避不攻击）
│  │  └─ 对家庭和子女有现实承诺
│  └─ 出现在目标伴侣也出现的地方
│
├─ 筛选伴侣 ⚠️ 比"追到"重要得多
│  ├─ 必问
│  │  ├─ 要不要孩子、什么时候要
│  │  ├─ 怎么看待钱、忠诚、父母、家务
│  │  └─ 冲突时：沟通、逃避、冷暴力还是攻击
│  ├─ 必观察
│  │  └─ 过去怎么处理承诺，亲密关系质量
│  └─ 必排除
│     └─ 成瘾、暴力、欺骗、拒担家庭责任
│
├─ 关系维护 ⚠️ 生育目标最常死在这里
│  ├─ 钱、性、父母、家务、孩子必须可讨论
│  ├─ 冲突后修复，不积累怨恨
│  └─ 关系长期摧毁你人生底线→止损离开
│
└─ 养育后代："生出来"不是终点
   ├─ 生之前：双方明确愿意+身体允许+收入撑得住
   └─ 生之后：给孩子安全感和秩序，投时间不只投钱
```

**Priority Path**

1. 排雷优先：违法、成瘾、高负债、健康崩溃、现金流断裂——任何一项归零
2. 找一个市场真正付钱的技能，尽快进入交易，别困在学习幻觉里
3. 用可靠交付积累信任，切断消耗型关系
4. 择偶竞争力建在"长期可依赖"上，不建在表面吸引力上
5. 筛选伴侣时优先看：生育意愿、责任感、金钱观、冲突处理方式、稳定性
6. 关系稳定后再进入生育——否则生育会变成长期风险放大器
